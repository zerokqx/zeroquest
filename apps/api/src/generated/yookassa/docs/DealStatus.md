# DealStatus

Статус сделки. Возможные значения: * opened — сделка открыта; можно выполнять платежи, возвраты и выплаты в составе сделки; * closed — сделка закрыта — вознаграждение перечислено продавцу и платформе или оплата возвращена покупателю; нельзя выполнять платежи, возвраты и выплаты в составе сделки.

## Enum

* `Opened` (value: `'opened'`)

* `Closed` (value: `'closed'`)
