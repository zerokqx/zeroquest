# PaymentStatus

Payment status. Possible values: pending, waiting_for_capture, succeeded, and canceled. More about the life cycle of a payment: https://yookassa.ru/developers/payment-acceptance/getting-started/payment-process#lifecycle

## Enum

* `Pending` (value: `'pending'`)

* `WaitingForCapture` (value: `'waiting_for_capture'`)

* `Succeeded` (value: `'succeeded'`)

* `Canceled` (value: `'canceled'`)
